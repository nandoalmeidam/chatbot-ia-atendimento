# Chatbot IA - Projeto

## Como rodar o projeto

### 1. Ativar ambiente virtual
No terminal digitar:

.venv\Scripts\activate


### 2. Subir backend
No terminal digitar:

uvicorn main:app --reload


### 3. Abrir frontend
- Abrir index.html com Live Server

## Importante
- Backend precisa estar rodando sempre
